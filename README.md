# Georgos Website

The premium, modern, responsive corporate website for **Georgos**, a respected family-owned retail business serving Pala, Kerala, India since 1963.

## Overview

This website serves as the digital presence of Georgos, communicating trust, heritage, family values, and community presence. It is a fully responsive React application built with Vite, emphasizing a premium aesthetic.

## Features

*   **Premium Design System:** Uses a custom heritage palette featuring Rich Charcoal, Deep Black, Luxury Gold, and Warm Cream.
*   **Responsive Layout:** Fully responsive across all devices (Desktop, Tablet, Mobile).
*   **Animations:** Smooth page transitions, glassmorphism effects, gradient hover states, and scroll-triggered animations using Framer Motion.
*   **Modern Tech Stack:** Built with React 19, Vite, React Router, Framer Motion, and Lucide React icons.
*   **Pages:**
    *   **Home:** Hero section with animated counters, store comparison, features, and heritage information.
    *   **Our Story (About):** Vertical timeline, leadership profiles, mission, vision, and core values.
    *   **Products:** Masonry layout showcasing product categories. (Note: Informational only, no online ordering).
    *   **Testimonials:** Animated review carousel and trust indicator badges.
    *   **Contact:** Store locations, interactive map placeholders, and a premium contact form.

## Tech Stack

*   **Frontend Framework:** React 19
*   **Build Tool:** Vite
*   **Routing:** React Router DOM v7
*   **Animations:** Framer Motion
*   **Icons:** Lucide React
*   **SEO:** React Helmet Async
*   **Styling:** Vanilla CSS with a comprehensive CSS variable design system

## Getting Started

### Prerequisites

*   Node.js (v18 or higher recommended)
*   npm (or yarn/pnpm)

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/N0-DE/Georgos-Website.git
    cd Georgos-Website
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

### Development Server

Run the local development server:

```bash
npm run dev
```

Open `http://localhost:5173` (or the port specified in your terminal) to view the application in your browser.

### Building for Production

To create a production-ready build:

```bash
npm run build
```

This will generate optimized static files in the `dist` directory, which can be deployed to any static hosting service (e.g., Vercel, Netlify, GitHub Pages).

To preview the production build locally:

```bash
npm run preview
```

## Branding Guidelines

The application strictly adheres to the Georgos branding:

*   **Colors:**
    *   Heritage Red: `#C41E24`
    *   Rich Charcoal: `#1C1917`
    *   Luxury Gold: `#C9A84C`
    *   Warm Cream: `#FDFBF7`
*   **Typography:** Outfit (Google Fonts)
*   **Aesthetic:** Premium, clean, and trustworthy, avoiding generic templates and excessive futuristic designs. Glassmorphism and subtle gradients are used to enhance the premium feel.
